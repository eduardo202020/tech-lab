# Guía rápida para capturas en documentación (/docs)

Sube las imágenes a la carpeta:

`public/docs/screenshots/`

Nombres sugeridos (ya referenciados en las páginas MDX):

- `inventario-lista.png`
- `inventario-detalle.png`
- `inventario-admin-form.png`
- `investigadores-lista.png`
- `investigadores-detalle.png`
- `investigadores-admin-form.png`
- `proyectos-lista.png`
- `proyectos-detalle.png`
- `proyectos-admin-form.png`
- `tecnologias-lista.png`
- `tecnologias-detalle.png`
- `tecnologias-relacion-proyectos.png`

Formato recomendado:

- PNG o WebP.
- Ancho entre 1400 y 2000 px.
- Captura en tema oscuro para consistencia visual.
- Ocultar datos sensibles antes de exportar.

Una vez copiadas las imágenes, se mostrarán automáticamente en `/docs` sin cambios adicionales de código.
