# Guía de Usuario de Tech Lab

Esta guía describe cómo usar la plataforma para consultar equipos, revisar información y gestionar préstamos según el tipo de usuario.

## ¿Qué encontrarás aquí?

- Acceso y autenticación.
- Uso del módulo de inventario.
- Flujo del módulo de investigadores.
- Flujo del módulo de proyectos.
- Flujo del módulo de tecnologías.
- Flujo general de préstamos.
- Preguntas frecuentes.

## Perfiles de usuario

- **Visitante:** puede explorar el catálogo de equipos.
- **Usuario autenticado:** puede revisar detalles y solicitar préstamos (según disponibilidad).
- **Administrador:** además puede crear, editar y eliminar equipos.
