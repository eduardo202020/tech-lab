# Tecnologías

Este módulo presenta el catálogo de tecnologías del Tech Lab y su conexión con proyectos activos.

## Flujo de uso (usuario)

1. Accede a **Tecnologías** para ver el catálogo.
2. Selecciona una tecnología para abrir su ficha.
3. Revisa descripción, casos de uso y alcance.
4. Consulta los proyectos donde se aplica esa tecnología.

## Flujo de exploración recomendada

1. Empieza por una tecnología de interés.
2. Abre los proyectos relacionados.
3. Desde el proyecto, identifica investigadores participantes.
4. Regresa al catálogo para comparar alternativas tecnológicas.

## Flujo de actualización de contenido (si aplica rol administrador)

1. Registrar o editar información de una tecnología.
2. Vincularla con proyectos relevantes.
3. Validar que la descripción sea clara y vigente.
4. Publicar y verificar enlaces cruzados.

## Buenas prácticas

- Usa descripciones orientadas a casos de uso reales.
- Evita información desactualizada en tendencias o herramientas.
- Mantén coherencia entre tecnología y proyectos vinculados.
