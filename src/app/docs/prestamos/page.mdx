# Préstamos

Esta sección resume el flujo general para solicitar y dar seguimiento a préstamos de equipos.

## Flujo básico

1. Revisa disponibilidad en inventario.
2. Selecciona un equipo prestable.
3. Registra la solicitud de préstamo.
4. Espera confirmación o validación administrativa.

## Buenas prácticas

- Verifica fechas de uso antes de solicitar.
- Incluye una descripción breve del propósito del préstamo.
- Devuelve el equipo en el plazo acordado.
