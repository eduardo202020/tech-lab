# Inventario

El módulo de inventario permite consultar equipos, su disponibilidad y datos técnicos.

## Búsqueda y filtros

- Usa la barra de búsqueda por nombre o texto clave.
- Aplica filtros por categoría, condición y disponibilidad.

## Ver detalle de un equipo

1. Haz clic sobre una tarjeta del inventario.
2. Se abrirá un modal con información completa: descripción, marca, modelo, serie, ubicación y especificaciones.

## Gestión (solo administrador)

- **Agregar:** crea un nuevo equipo con sus campos principales.
- **Editar:** actualiza datos de un equipo existente.
- **Eliminar:** retira un equipo del inventario.
