# Acceso y autenticación

## Ingreso a la plataforma

1. Abre la aplicación en tu navegador.
2. Haz clic en **Iniciar sesión**.
3. Completa el flujo de autenticación.

## ¿Qué cambia al iniciar sesión?

- Se habilitan acciones de usuario autenticado.
- Si tienes rol de administrador, verás opciones de gestión (agregar/editar/eliminar).

## Recomendaciones

- Usa tu correo institucional para mantener trazabilidad.
- Si el acceso falla, verifica sesión activa y vuelve a autenticar.
