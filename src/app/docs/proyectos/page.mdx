# Proyectos

El módulo de proyectos centraliza la planificación, estado, prioridad, equipo participante y tecnologías utilizadas.

## Flujo de uso (usuario)

1. Entra a **Proyectos** para ver el listado general.
2. Usa filtros por estado, prioridad o categoría para reducir resultados.
3. Abre un proyecto para revisar su ficha completa.
4. Consulta objetivos, avances, responsables y enlaces de referencia.

## Flujo dentro del detalle de proyecto

1. Revisa estado y progreso actual.
2. Verifica investigadores vinculados.
3. Consulta tecnologías relacionadas.
4. Si existen, abre enlaces externos (demo, repositorio, documentación).

## Flujo de administración (si aplica rol administrador)

1. Crear un nuevo proyecto con datos base.
2. Asignar investigadores y tecnologías relacionadas.
3. Actualizar progreso, prioridad y estado según avance.
4. Publicar cambios y confirmar consistencia de relaciones.

## Buenas prácticas

- Define claramente objetivo, alcance y fechas.
- Mantén el progreso actualizado para seguimiento del equipo.
- Evita estados ambiguos para facilitar reportes.
