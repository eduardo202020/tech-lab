# Investigadores

Este módulo permite consultar perfiles de investigadores, sus especialidades y su relación con proyectos.

## Flujo de uso (usuario)

1. Ingresa a la sección **Investigadores**.
2. Usa búsqueda o filtros para ubicar personas por nombre, especialidad o estado.
3. Revisa la tarjeta resumen para validar información clave.
4. Abre el detalle del investigador para ver perfil completo y proyectos asociados.

## Flujo de navegación recomendada

1. Desde un investigador, abre un proyecto relacionado.
2. Desde el proyecto, revisa tecnologías vinculadas.
3. Vuelve al perfil para comparar colaboración entre equipos.

## Flujo de administración (si aplica rol administrador)

1. Crear un nuevo registro de investigador.
2. Editar datos del perfil (contacto, líneas de trabajo, estado).
3. Actualizar relación con proyectos.
4. Guardar cambios y validar visualización pública.

## Buenas prácticas

- Mantén nombres y correos institucionales actualizados.
- Evita duplicar perfiles del mismo investigador.
- Revisa que los proyectos vinculados estén vigentes.
